# Darkboot

Dark theme for Bootstrap 4, css (darkboot.css) is located in dist directory, but if you need source it is inside scss directory named darkboot.scss :)

![Darkboot image](img/darkboot.png?raw=true "Darkboot")


## Getting Started

You can either use provided css from dist directory or build and customize your own build, source scss (darkboot.scss) file is in scss directory.

### Prerequisites

```
sass
```

### Installing

To build your own css run that

```
sass scss/darkboot.scss dist/darkboot.css
```


## Built With

* [Bootstrap](https://github.com/twbs/bootstrap) - Bootstrap 4
